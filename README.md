####Steps to make it work: 
Clone the project: 

`$ git clone git@github.com:540/web.git`

`$ git clone https://github.com/540/web.git`
  
Navigate to the project folder

1. `$ npm install`

2. `$ bower install`

3. `$ node build`

Deploy to GitHub Pages

`git subtree push --prefix dist origin gh-pages`
